LeetCode18-四数之和

发表于 2018-11-19 | 分类于 LeetCode

题目链接

英文链接：https://leetcode.com/problems/4sum/

中文链接：https://leetcode-cn.com/problems/4sum/

题目详述

给定一个包含 n 个整数的数组 nums 和一个目标值 target，判断 nums 中是否存在四个元素 a， b，c 和 d ，使得 a + b + c + d 的值与 target 相等？找出所有满足条件且不重复的四元组。

注意：

答案中不可以包含重复的四元组。

题目详解

LeetCode15-三数之和 现在变成了四数之和，同样的思想运用双指针。

  • 首先把数组从小到大进行排序。
  • 遍历过程中固定两个元素。
  • 再运用双指针的思想，一个从左往右扫描，一个从右往左扫描。
