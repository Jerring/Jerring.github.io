题目详述

给一非空的单词列表，返回前 k 个出现次数最多的单词。

返回的答案应该按单词出现频率由高到低排序。如果不同的单词有相同出现频率，按字母顺序排序。

注意：

  1. 假定 k 总为有效值， 1 ≤ k ≤ 集合元素数。
  2. 输入的单词均由小写字母组成。

扩展练习：

  1. 尝试以 O(n log k) 时间复杂度和 O(n) 空间复杂度解决。

题目详解

堆的运用。

  • 首先运用 Map 统计单词出现次数。
  • 求前K个高频单词，构建一个小顶堆。
  • 遍历 map 的 ketSet，加入小顶堆并保持堆的大小为 k。
  • 最后一个一个取出堆中的元素。
  • 注意由于是小顶堆，最先取出的元素是最后一个元素，要翻转结果链表。
