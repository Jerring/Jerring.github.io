题目详述

给定一个整数数组和一个整数 k, 你需要在数组里找到不同的 k-diff 数对。这里将 k-diff 数对定义为一个整数对 (i, j), 其中 i 和 j 都是数组中的数字，且两数之差的绝对值是 k.

注意:

  1. 数对 (i, j) 和数对 (j, i) 被算作同一数对。
  2. 数组的长度不超过10,000。
  3. 所有输入的整数的范围在 [-1e7, 1e7]。

题目详解

  • 先对数组进行排序，再运用双指针进行扫描。
  • 指针 i 和 j 分别指向前一个和后一个元素，初始化为 0 和 1。
  • 当元素差小于 k 时，j 右移。
  • 当元素差大于 k 时，i 右移。
  • 当元素差等于 k 时，添加这个数对到结果中，i 右移，j 右移，并跳过重复元素。
  • 当 i 追上 j 时，j 右移。
  • 重复这个过程直至迭代结束。
  • 时间复杂度为 O(nlogn)。
  • 也可以运用 HashMap 统计各个元素出现的次数，再进行查找是否存在满足条件的另一个元素。
  • 时间复杂度为 O(n)。
