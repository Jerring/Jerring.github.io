LeetCode463-岛屿的周长

发表于 2019-05-02 | 分类于 LeetCode

题目链接

英文链接：https://leetcode.com/problems/island-perimeter/

中文链接：https://leetcode-cn.com/problems/island-perimeter/

题目详述

给定一个包含 0 和 1 的二维网格地图，其中 1 表示陆地 0 表示水域。

网格中的格子水平和垂直方向相连（对角线方向不相连）。整个网格被水完全包围，但其中恰好有一个岛屿（或者说，一个或多个表示陆地的格子相连组成的岛屿）。

岛屿中没有“湖”（“湖” 指水域在岛屿内部且不和岛屿周围的水相连）。格子是边长为 1 的正方形。网格为长方形，且宽度和高度均不超过 100 。计算这个岛屿的周长。

示例 :

题目详解

  • 每碰到一个陆地，数存在的边。
  • 有两种方式，一种直接相加满足条件的的情况，另一种先加 4，再减去不满足条件的情况。

方法一：直接相加（边缘或相邻是水域）。

方法二：先加后减（上边或左边是陆地）
