LeetCode1033-移动石头直到连续

发表于 2019-06-18 | 分类于 LeetCode

题目详述

三枚石子放置在数轴上，位置分别为 a，b，c。

每一回合，我们假设这三枚石子当前分别位于位置 x, y, z 且 x < y < z。从位置 x 或者是位置 z 拿起一枚石子，并将该石子移动到某一整数位置 k 处，其中 x < k < z 且 k != y。

当你无法进行任何移动时，即，这些石子的位置连续时，游戏结束。

要使游戏结束，你可以执行的最小和最大移动次数分别是多少？ 以长度为 2 的数组形式返回答案：answer = [minimum_moves, maximum_moves]

提示：

  1. 1 <= a <= 100
  2. 1 <= b <= 100
  3. 1 <= c <= 100
  4. a != b, b != c, c != a

题目详解

  • 注意 a、b、c 不一定有序，我们需要自己得到有序的 x、y、z。
  • 最大值很好计算，x、z 每次移动 1 步，直到石子连续，移动次数为 z - x - 2。
  • 对于最小值，容易知道它最大为 2，它还存在为 0 和为 1 的情况。
    • 0：x、y、z 连续，即 y - x == 1 && z - y == 1。
    • 1：x、y 连续或 y、z 连续，但不是全部连续（如1、2、5）；x、y 相隔 2 或 y、z 相隔 2（如 1、3、7，可以把 7 移动 2，移动次数为 1）。综合起来就是 y - x <= 2 || z - y <= 2。
    • 2：x、y 相隔大于 2，y、z 相隔大于 2。
