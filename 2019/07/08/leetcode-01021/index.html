LeetCode1021-删除最外层的括号

发表于 2019-07-08 | 分类于 LeetCode

题目链接

英文链接：https://leetcode.com/problems/remove-outermost-parentheses/

中文链接：https://leetcode-cn.com/problems/remove-outermost-parentheses/

题目详述

有效括号字符串为空 (“”)、”(“ + A + “)” 或 A + B，其中 A 和 B 都是有效的括号字符串，+ 代表字符串的连接。例如，””，”()”，”(())()” 和 “(()(()))” 都是有效的括号字符串。

如果有效字符串 S 非空，且不存在将其拆分为 S = A+B 的方法，我们称其为原语（primitive），其中 A 和 B 都是非空有效括号字符串。

给出一个非空有效字符串 S，考虑将其进行原语化分解，使得：S = P_1 + P_2 + … + P_k，其中 P_i 是有效括号字符串原语。

对 S 进行原语化分解，删除分解中每个原语字符串的最外层括号，返回 S 。

题目详解

  • 根据定义，原语字符串是有效的并且它的左括号个数和右括号个数相等。
  • 去掉最外层的括号就是去掉最外层配对的括号。
  • 遍历字符串，运用一个变量 cnt 进行统计。
    • 如果是 '('，执行 ++ 操作，如果 cnt > 1 说明它是内层括号，添加到结果中。
    • 如果是 ')'，执行 -- 操作，如果 cnt > 0 说明它是内层括号，添加到结果中。
  • 也可以直接把整个内层括号字符串添加到结果中。
  • 遍历字符串，运用一个变量 cnt 进行统计，执行与上述类似的操作。如果 cnt == 0，说明最外层括号配对成功，直接把内层括号字符串添加到结果中。
